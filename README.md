# SBGATOOLS 使用手册

> **当前版本**: v1.4.3+101
> 一款用于 maimai DX 游戏的功能票发放与存档修改工具。

## 快速开始

### 1. 导入许可证

首次使用需要导入许可证文件（`.sbl` 格式）：

1. 打开应用，进入**设置**页面
2. 点击**导入许可证**按钮
3. 选择许可证文件

> ⚠️ 许可证文件创建后 **1 小时内** 必须导入，超时将无法使用

### 2. 配置机厅信息

在设置页面配置机厅信息（区域、机厅名称、客户端ID等），使用线下常去的机厅信息可降低账号异常概率。

> ⚠️ 客户端 ID 为必填项，格式为 11 位字符

### 3. 开始使用

应用底部有两个标签页：

- **发送功能票**：扫描二维码 → 自动登录 → 发送功能票 → 自动登出
- **获取账号数据**：扫描二维码 → 自动登录 → 获取完整数据 → 配置自定义数据 → 上传存档 → 自动登出

操作步骤：
1. 点击**扫描二维码**或**手动输入**
2. 扫描/输入公众号生成的二维码（支持自动读取剪贴板）
3. 等待自动处理完成（处理中显示进度和实时日志）
4. 查看日志确认结果（支持导出诊断日志）

## 功能说明

### 时间同步状态

主界面左上角显示时间同步状态图标：

| 图标 | 颜色 | 说明 |
| -- | -- | -- |
| ✓ | 🟢 绿色 | NTP 同步成功，已获得可信网络时间 |
| ⏱ | 🟠 橙色 | 正在进行 NTP 同步或尚未检测 |
| ✗ | 🔴 红色 | NTP 同步失败，点击重新同步 |

> 💡 应用启动时自动检测。NTP 会解析服务器原始时间响应并过滤高延迟/异常样本；同步期间不能点击执行或扫描二维码，实际执行前还会强制确认可信网络时间仍在新鲜期内。同步成功后点击图标可查看当前可信时间和偏移值。

### 二维码处理

支持**扫描二维码**（需相机权限）和**手动输入**（支持自动读取剪贴板）两种方式。

> 二维码有效期为 **10 分钟**，超时需重新获取

### 功能票倍数

仅在「发送功能票」标签页可调整：

| 倍数 | 价格 | 安全性      |
| -- | -- | -------- |
| 2倍 | 1  | ✅ 安全     |
| 3倍 | 2  | ✅ 安全（默认） |
| 5倍 | 4  | ⚠️ 危险    |

> ⚠️ 5 倍功能票存在风险，请谨慎使用
> ⚠️ 4 倍和 6 倍功能票因游戏版本 1.55 起不再支持，已移除

### 发票等待时间

固定 20 秒等待，无需配置。计时从 **UserLoginApi 响应成功返回** 后开始，获取用户数据和库存检查会在等待窗口内完成，上传前只补足剩余等待时间。

### 许可证系统

- **单一用户许可证**：仅限指定 UserID 使用
- **管理员许可证**：不限制 UserID
- 支持永久有效和限时有效两种有效期
- 创建后 1 小时内必须导入
- 执行时会在二维码解析后验证许可证、用户权限和可信网络时间新鲜度

### 自动更新

进入**设置**页面点击**检查更新**，也可开启**自动检查更新**（每次打开应用时自动检测新版本）。

### 更新公告

版本更新后首次打开应用时，会自动弹窗展示该版本的更新内容。

### 获取账号数据

第二个标签页提供完整的存档修改功能。标签页顶部有"修改并上传"开关（默认开启）：

- **开启**：获取数据 → 配置修改 → 上传存档 → 登出
- **关闭**：获取数据 → 直接登出（仅更新本地数据，不修改存档）

存档上传流程已按 SDGB 1.55 DLL 等价逻辑迁移：登录后构建 `UserLog` 与 `UserAll`，先上传 `GetUserNewItemListApi`，再上传 `UpsertUserAllApi`。上传存档的 60 秒等待同样从 UserLoginApi 响应成功返回后开始计时；等待结束前会刷新实际上传时刻相关字段。

#### 可自定义字段

数据获取完成后自动弹出配置页面，每个配置项旁显示当前原始值：

| 字段 | 说明 | 特殊行为 |
| ---- | ---- | -------- |
| 临时 Rating | 临时显示指定 Rating | 上机游玩一把后自动还原 |
| 累计游玩次数 | 修改总游玩次数 | 上传时自动 +1 |
| 当前版本游玩次数 | 修改当前版本游玩次数 | 上传时自动 +1 |
| 舞里程 | 将舞里程修改为目标值 | 开启后填写修改后的目标值；默认启用格式化，尾数非0/9归0且封顶99999；关闭格式化后不限制尾数和上限 |
| 增加礼物数量 | 购买礼物增加库存 | 原库存 + 增加数量；记录购买数量，不扣减舞里程 |
| 清票 | 清零功能票库存 | 自动判断（有票启用，无票禁用），显示倍数详情 |
| 歌曲解锁 | 解锁需要特定条件的歌曲 | 三种模式：手动/部分/全部 |
| KaleidxScope解锁 ⚠️ | 解锁万花筒区域的门与钥匙 | 选择区域后开启发现门/获取钥匙，已持有自动锁定 |
| 相同旅行伙伴 | 单推旅行伙伴 | 5个位置设为相同队长角色 |
| 旅行伙伴升级 | 旅行伙伴升级 | 修改至9999级觉醒6 |
| 发放旅行伙伴 | 添加指定旅行伙伴 | 选取窗口支持图片预览+搜索+多选，已有伙伴自动跳过 |
| 友人对战 ⚠️ | 修改对战段位和CP积分，可覆盖历史最高段位 | CP 输入限制为所选段位可真实存在的范围，可覆盖历史最高段位（开关控制） |
| 移动距离 | 设为999KM | 可在地图中使用移动距离获取道具 |
| 收藏品 ⚠️ | 获取或移除收藏品 | 支持头像/姓名框/背景/称号/搭档，窗口选取+搜索+多选 |
| 歌曲成绩上传 | 覆盖上传自定义歌曲成绩，显示封面图和原始值供参考 | 至多3首；通过选歌器选择歌曲，难度只显示实际存在谱面，DX分数范围按谱面物量限制；1010000分时可写理论次数 |

> ⚠️ 标有 ⚠️ 的功能存在风险，请谨慎使用
> ⚠️ 自定义数据1次执行内有效，执行完成后自动清除
> ⚠️ 数值不能为负数

### 数据页面

第三个标签页提供账号数据浏览和收藏品/歌曲查询：

- **账号页**：展示用户信息、DX Rating、游玩统计、当前装备
- **收藏品查询**：按种类浏览头像、姓名框、背景、称号、搭档的ID、名称和图片，支持按持有状态筛选
- **歌曲查询**：按分类（标准/DX/宴会会场）和版本筛选歌曲，查看歌曲详情

### 日志系统

- 实时显示操作日志，支持四种级别（信息/成功/警告/错误）
- 点击**导出**按钮可保存完整诊断日志（含操作日志、网络通信详情、上传数据校验）

### 取消操作

处理过程中可点击**取消**按钮，应用会在下一个安全检查点跳转到登出环节，确保账号安全退出。

### 后台防杀

执行期间自动启动前台服务，切换到后台或小窗状态不会中断流程。状态栏显示常驻通知，通知上的"取消"按钮可随时取消操作。

### 异常退出自动登出

如果应用在执行过程中意外关闭（如系统杀进程），下次检测到该账号已登录时，会自动使用保存的会话信息尝试登出，确保账号安全。

## 常见问题

### Q: 提示"二维码已过期"

**A:** 二维码有效期为 10 分钟，请在公众号上获取新的二维码。

### Q: 提示"许可证不允许操作此用户ID"

**A:** 当前许可证未包含您的 UserID，请联系管理员获取授权或使用管理员许可证。

### Q: 提示"许可证导入时效已过"

**A:** 许可证文件创建后 1 小时内必须导入，超时请联系管理员补发新许可证。

### Q: 提示"许可证已过期"

**A:** 许可证已超过有效期，请联系管理员续期或重新获取。

### Q: 提示"可信网络时间不可用"

**A:** NTP 同步失败或可信网络时间已过期。请检查网络后重新执行；同步期间应用会禁止开始执行。

### Q: 发票失败

**A:** 可能原因：

- 账号内已存在功能票（应用会自动跳过）
- IP 地址被服务器限制
- 网络连接问题

### Q: 存档发送失败

**A:** 可能原因：

- 网络连接不稳定（应用会自动重试最多 3 次）
- IP 地址被封禁

### Q: 扫描二维码不工作

**A:** 请在系统设置中授予应用相机权限。

### Q: 无法安装更新

**A:** Android 8.0+ 需要授予"安装未知应用"权限，应用会在下载完成后提示授权。

### Q: 检查更新/下载失败

**A:** 可能是网络问题或 GitHub 访问受限，请尝试使用代理或 VPN。

### Q: "获取账号数据"中的自定义数据是什么意思？

**A:** 数据获取完成后会自动弹出配置页面，您可以临时修改游玩数据。每个配置项旁显示当前原始值供参考：

| 字段 | 效果 |
| ---- | ---- |
| 临时 Rating | 临时显示指定 Rating，上机游玩一把后自动还原 |
| 游玩次数 | 修改累计/当前版本游玩次数 |
| 舞里程 | 修改为目标舞里程值；默认格式化为合理尾数并封顶99999 |
| 增加礼物数量 | 购买礼物增加库存，通过送礼提升伙伴好感度；不扣减舞里程 |
| 清票 | 有票时自动清零，避免上传失败 |
| 歌曲解锁 | 解锁需要特定条件的歌曲 |
| KaleidxScope解锁 ⚠️ | 解锁万花筒区域的门与钥匙 |
| 旅行伙伴 | 单推/升级/发放旅行伙伴 |
| 友人对战 ⚠️ | 修改对战段位和CP积分，可覆盖历史最高段位；CP按段位真实范围限制 |
| 收藏品 ⚠️ | 获取或移除头像/姓名框/背景/称号/搭档 |
| 歌曲成绩上传 | 通过选歌器覆盖上传自定义歌曲成绩，难度和DX分数范围按实际谱面限制 |

> 配置1次执行内有效，执行完成后自动清除

### Q: "修改并上传"开关有什么作用？

**A:** 该开关位于"获取账号数据"标签页顶部，默认开启：

- **开启**：获取数据后弹出配置页供您修改，确认后上传存档
- **关闭**：获取数据后直接登出，仅更新本地数据，不修改服务器存档

### Q: 日志导出文件在哪里？

**A:** 日志导出时会弹出文件选择对话框，您可以选择保存位置。导出文件为 `.txt` 格式。

### Q: 处理过程中可以取消吗？

**A:** 可以。点击**取消**按钮后，应用会在下一个安全检查点跳转到登出环节，确保账号安全退出。

## 注意事项

1. **二维码时效**：有效期 10 分钟，请及时使用
2. **许可证时效**：创建后 1 小时内必须导入
3. **功能票倍数**：5 倍存在风险，建议使用默认的 3 倍（4 倍和 6 倍因游戏 1.55 版本起不再支持，已移除）
4. **发票等待**：固定 20 秒等待，从登录响应成功返回后计时
5. **网络环境**：确保网络连接稳定，如需代理请在系统 WLAN 设置中配置
6. **机厅配置**：使用真实的机厅信息可降低账号异常风险
7. **时间同步**：启动时自动 NTP 检测，可信时间由应用自动维护（独立于系统时间，无需手动同步系统时间）；执行前必须拥有新鲜可信网络时间，同步期间不能开始执行
8. **存档修改**：请谨慎使用，配置时登录态保持中，请尽快完成配置；保存后若未到上传边界会继续等待，到时自动上传
9. **安全登出**：无论操作成功或失败，应用都会尝试安全登出；异常退出后下次检测到已登录会自动尝试登出
