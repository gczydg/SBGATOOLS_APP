# SBGATOOLS 使用教程

一款用于某游戏的功能票发放工具。

## 快速开始

### 1. 导入许可证

首次使用需要导入许可证文件（`.sbl` 格式）：

1. 打开应用，进入**设置**页面
2. 点击**导入许可证**按钮
3. 选择许可证文件

> ⚠️ 许可证文件创建后 **1 小时内** 必须导入，超时将无法使用

### 2. 配置机厅信息
在设置页面配置以下信息：

| 配置项 | 说明 |
|--------|------|
| 区域 ID | 机厅所在区域编号 |
| 区域名称 | 区域名称 |
| 机厅 ID | 机厅编号 |
| 机厅名称 | 机厅名称 |
| 功能票倍数 | 发票倍数（2-6倍） |

使用线下常去的机厅信息，降低账号异常概率

### 3. 开始使用

1. 返回主页，点击**扫描二维码**或**手动输入**
2. 扫描/输入 公众号 生成的二维码
3. 等待自动处理完成

## 功能说明

### 二维码处理

支持两种方式：
- **扫描二维码**：使用相机扫描二维码
- **手动输入**：直接粘贴二维码文本内容

> 二维码有效期为 **10 分钟**，超时需重新获取

### 功能票倍数

| 倍数 | 价格 | 安全性 |
|------|------|--------|
| 2倍 | 1 | ✅ 安全 |
| 3倍 | 2 | ✅ 安全（默认） |
| 4倍 | 3 | ⚠️ 危险 |（1.53后未测试）
| 5倍 | 4 | ⚠️ 危险 |（1.53后未测试）
| 6倍 | 5 | ⚠️ 危险 |（1.53后未测试）

> ⚠️ 4/5/6 倍功能票存在风险，请谨慎使用

### 许可证类型

| 类型 | 说明 |
|------|------|
| 单一用户许可证 | 仅限指定 UserID 使用 |
| 管理员许可证 | 不限制 UserID |

## 操作流程

```
扫描二维码 → 解析用户信息 → 自动登录 → 检查功能票 → 发票 → 自动登出
```

整个过程自动完成，您只需：
1. 提供有效的二维码
2. 等待处理完成
3. 查看日志确认结果

## 常见问题

### Q: 提示"二维码已过期"
**A:** 二维码有效期为 10 分钟，请在公众号上获取新的二维码。

### Q: 提示"许可证不允许操作此用户ID"
**A:** 当前许可证未包含您的 UserID

### Q: 提示"许可证导入时效已过"
**A:** 许可证文件创建后 1 小时内必须导入, 超时请找我补发

### Q: 发票失败
**A:** 可能原因：
- 账号内已存在功能票
- IP 地址被服务器限制
- 网络连接问题

### Q: 扫描二维码不工作
**A:** 请在系统设置中授予应用相机权限。

## 注意事项

1. **二维码时效**：二维码有效期 10 分钟，请及时使用
2. **许可证时效**：许可证文件创建后 1 小时内必须导入

4. **网络环境**：确保网络连接稳定
